# Global Superstore Sales Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Global Superstore dataset to uncover business insights related to sales, profit, customer behavior, products, and regional performance.

## Objectives

- Analyze sales by category
- Analyze profit by category
- Identify top-selling products
- Identify most profitable products
- Study regional sales
- Study customer segments
- Analyze shipping modes
- Examine Discount vs Profit
- Analyze monthly sales trends

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Kaggle

## Key Insights

- Technology generated the highest sales and profit.
- Phones had the highest sales among sub-categories.
- Copiers generated the highest profit.
- Tables produced negative profit.
- Consumer customers generated the highest revenue.
- The United States was the highest-selling country.
- Standard Class was the most commonly used shipping mode.
- Higher discounts were associated with lower profit.
- Apple Smart Phone generated the highest sales.
- Canon imageCLASS 2200 Advanced Copier generated the highest profit.
